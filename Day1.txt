Difference between HTTP1.1 vs HTTP2

Http1.1 works on textual format where http2 works on binary protocol.
In http1.1 head of line blocking that blocks all the requests behind it until it doesn’t get its al resources, in http2 It allows multiplexing so one TCP connection is required for multiple requests
Unlike hhtp1.1,  http2’s ability to transmit more data per client-server communication cycle greatly improves web performance
Page load speed is better than http1.1 in http2
